# Contributing

Thanks for helping improve the public resource radar template.

Good contributions include:

- clearer docs for first-time users;
- better public-safe demo records;
- safer validation checks;
- improved scoring or lifecycle examples;
- downstream lane taxonomy refinements;
- bug fixes in deterministic report generation.

Please keep contributions public-safe. Do not include private bookmarks,
credentials, local paths, private notes, account state, or non-public candidate
pools.
